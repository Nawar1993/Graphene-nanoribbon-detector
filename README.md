This file includes software codes and graphics for modeling and simulating a graphene ribbon detector in the mid-infrared range.
A width of 22 nm was chosen for the graphene ribbon, which in turn led to the formation of resonant absorption peaks between 3 μm and 8 μm.
We plotted the photocurrent, dark current, and responsivity.
The normalization curves show that the thermal sensitivity of the graphene ribbon detector decreases by small percentages, whereas the sensitivity of the silicon detector would collapse by more than 99 percent, which necessitates continuous cooling
